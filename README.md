# Alice City Guesser

Навык для Яндекс.Алисы - игра "Угадай город по фотографии". Пользователю показываются фотографии городов, и он должен угадать их названия.

## О проекте

Интерактивная игра для Яндекс.Алисы, где игроку показываются фотографии городов, а он должен угадать их названия. Идеально подходит для проверки знаний географии и развлечения в свободное время.

## Технологический стек

- **Backend**: Python 3.8+
- **Фреймворк**: Flask 2.3.3
- **ORM**: SQLAlchemy 2.0.19
- **База данных**: SQLite
- **Интеграция**: Яндекс.Диалоги API
- **Сервер**: WSGI (Gunicorn для production)
- **Контейнеризация**: Docker (опционально)

## Возможности

- 30+ городов со всего мира
- По 2 фотографии на каждый город для подсказок
- Система очков с отслеживанием прогресса
- Несколько попыток угадывания (до 3 фотографий на один город)
- Персонализация - Алиса запоминает имя пользователя
- Таблица лидеров с использованием SQLite

## Доступные города

Москва, Нью-Йорк, Париж, Дубай, Абу-Даби, Стамбул, Бангкок, Доха, Шарджа, Тель-Авив, Анталья, Дели, Каир, Иерусалим, Пхукет, Паттайя, Хургада, Маскат, Бали, Осака, Сингапур, Барселона, Пальма-де-Мальорка, Милан, Гонконг, Мекка, Лондон, Куала-Лумпур, Токио, Сеул, Макао
